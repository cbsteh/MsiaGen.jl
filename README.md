
# MsiaGen: Stochastic Daily Weather Generator for Malaysia (DRAFT)

## Overview
MsiaGen generates daily weather data (minimum and maximum air temperature, wind speed, and rainfall) for Malaysia. It has been tested and validated across 23 locations in Peninsular Malaysia and East Malaysia (Sabah and Sarawak).

More details coming up, upon publication of MsiaGen in a journal.
